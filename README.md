# cricclubs-scraper

This repository contains multiple CricClubs scraper scripts for scraping match and series data into Supabase.

## Available scrapers

- `ball_by_ball_scraper.py` — scrape ball-by-ball score data
- `match_info_scraper.py` — scrape match information and metadata
- `scorecard_scraper.py` — scrape batting and bowling scorecards
- `player_info_scraper.py` — scrape player profile data from CricClubs
- `player_info_scraper.py` — scrape player profile data from CricClubs
